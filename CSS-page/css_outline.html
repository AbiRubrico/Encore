CSS Outline

A outline is a line drawn around elements, outside the boundaries for the "stand - out" element.

CSS has the following outline properties:

  • outline-style
  • outline-offset
  • outline-width
  • outline-color
  • outline

    • Outline Color

    The outline-color property is used to set the color of the outline.

    Example: (Try editing it in your css file)

    p {
  border: 1px solid black;
  outline-style: solid;
  outline-color: green;
}

    Outline Width

    The outline-width property specifies the width of the outline, here are the following values:

  • thin (typically 1px)
  • medium (typically 3px)
  • thick (typically 5px)
  • A specific size (in px, pt, cm, em, etc)

    • Example: (Try editing it in your css file)

    p.ex1 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: green;
  outline-width: thin;
}
