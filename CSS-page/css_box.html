CSS Box Model

The CSS box model is basically a case that folds over each HTML component. It includes of: margins, borders, padding, and the actual content.

Explanation of the different parts:

  • Content - The content of the box, where text and images appear.
  • Padding - Clears an area around the content and its padding is transparent.
  • Margin - Clears an area outside the border and its margin is transparent.
  • Border - A border that goes around the padding and content.

    • Example: (Try editing it in your css file)

    div {
  width: 400px;
  border: 30px solid aliceblue;
  padding: 20px;
  margin: 20px;
}
