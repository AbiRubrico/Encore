CSS Opacity

CSS opacity refers to the transparency of an element..

Opacity property can take from 0.0 to 1.0 When the value is lower, the transparency is higher.

Example: (Try editing it in your css file)

img { opacity: 0.5; filter: alpha(opacity=50); /* For IE8 and earlier */ } img:hover { opacity: 1.0; filter: alpha(opacity=100); /* For IE8 and earlier */ }
