CSS Float

Float is a property used to specify whether an element should float or not and in what area it should be float.

left - the element floats on the left side.

right - the element floats on the right side.

noneit removes the float property from the element.

Example: (Try editng it in your css file)
 
        img {
               float: left;
            }
