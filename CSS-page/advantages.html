What are the advantages of CSS?

CSS saves time − The developer can compose CSS once and afterward reuse same sheet in various HTML pages. The developer can characterize a style for every HTML component and apply it to the same number of Web pages as you need.

Pages load faster − On the off chance that you are utilizing CSS, you don't have to compose HTML label characteristics each time. Simply keep in touch with one CSS guideline of a tag and apply it to every one of the events of that tag. So less code implies quicker download times.

Easy maintenance − To make a global change, simply change the style, and all elements in all the web pages will be updated automatically.

Multiple Device Compatibility

Global web standards − Now HTML attributes are being deprecated and it is being recommended to use CSS in all the HTML pages to make them compatible to future browsers.
