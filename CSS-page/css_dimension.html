CSS Dimension

It is a property used for controlling an element's height and width.

Width and Height Properties

It is a control property that defines the width and height of an element's content area. It can take the values of a length "px, pt , em, etc," percentage, or an auto to control element's height and width.

Example: (Try editng it in your css file)
 div {
    width: 300px;
    height: 200px;
}

Max-Height and Max-Width

max-height and max-width property allows you to specify the maximum limitation of the height and width of a block.

Example: (Try editng it in your css file)
 div {
    max-height: 300px;
    max-width: 500px;
}

Min-Height and Min-Width

min-height and min-width property allows you to specify the minimun limitation of the height and width of a block.

Example: (Try editng it in your css file)
 div {
    min-height: 300px;
    min-width: 500px;
}
