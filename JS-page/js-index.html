JavaScript Introduction

JavaScript (formerly known as LiveScript) is a Web programming language which is often used on creating interactive effects with browsers. It is an object-oriented, functional, and imperative scripting language. JavaScript allows you to do things which include declaring variables, storing values, defining functions and classes, using external modules, writing event handlers, and more.

It has been standardized as ECMAscript by ECMA International.

Null, this is supposedly considered to be “nothing” but in JavaScript, Null is used to define an object is empty. What is the difference between Null and Undefined? A Null is still an object even if it has no value, undefined is still undefined whether having a value or not.

JS Array Iteration

Array.forEach()
This is a callback function for each element inside a array.
Ex.


Array.map()
This method produces another array to perform the function given on each array element, this method will not execute if there are no values inside the array.
Ex.


Array.filter() / every() / some()
The filter() method produces and stores elements in another array that meets the standards of a function given. every() almost do the same functionalities, except, it returns a boolean value and checks if all elements inside the array complies with the given function. some() also has the same functionalities with every() but it only concerns other elements in the array.
Ex.


Array.reduce()
This method produces a value that consists of all the elements inside the array, this method follows a left-to-right rule when complying to a function. reduceRight() is the same as reduce() but it follows a right-to-left rule.
Ex.


Array.indexOf() / lastIndexOf() / find()
This indexOf() method looks for an element and returns the position of that element. lastIndexOf() searches an element from the end of the array. find() looks for an element that passes the requirements of the function given.

JS Dates

Declaring "date" objects in JavaScript are done through the following:

1. new Date() - this declares the current date.
2. new Date(year, month, day, hours, minutes, seconds, milliseconds) - this declares a specified date and time.
3. new Date(milliseconds) - this manipulates the date by increasing or decreasing the given millisecond to the date provided and calculate it to a specified date.
4. new Date(dateString) - this manipulate the date object by inserting a specified date [ex: new Date("April 21, 1997 11:59:00")]

case a:
// condition
break;

case b:
// condition
break;

default:

}

JS Loop For


The For Loop

Syntax:
for (statement1; statement2; statement3) {
// code to be executed
}

statement1 is only executed once (ex: i = 0)
statement2 is the condition (ex: x = y.length)
statement3 is executed after the code has been executed (ex: i++ / i--)

For/In Loop

Syntax:
for (args) {
// code to be executed
}
