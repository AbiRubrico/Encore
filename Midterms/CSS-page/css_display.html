CSS Display

It is a property that controls the container's sort created by a component.

The CSS detail characterizes the default show an incentive for every one of the components. The ordinarily utilized are block and inlines.

Display Block

The " block " value is property which forces an element to behave like block-level element.

Example: (Try editing it in your css file)

 span {
    display: block;
}
a {
    display: block;
}

Display Inline

The " inline " value is a property which causes an element to behave as though it were an inline-level element.

Example: (Try editng it in your css file)

 p {
    display: inline;
}
ul li {
    display: inline;
}

Display Inline-Block

The " inline-block " value is a property which causes an element to generate a block box that will be flowed with surrounding content i.e. in the same line as adjacent content.

Example: (Try editng it in your css file)

 div {
    display: inline-block;
}
span {
    display: inline-block;
}

Display None

The " none " value is a property which simply generates no boxes at all or nothing.

Example: (Try editng it in your css file)

 h1 {
    display: none;
}
p {
    display: none;
}
