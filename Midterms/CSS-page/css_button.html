CSS Buttons

HTML buttons can be styled using CSS. You can assign color, background color, text-align, font-size, width & height, border, border-radius, padding, margin, float, display and many more CSS styling properties for your button elements to give them a professional look and set their positioning.

.button {
  background-color: #4CAF50; /* Green */
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
}
