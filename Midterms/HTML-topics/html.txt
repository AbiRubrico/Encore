
HTML History

*Hyper Text Markup Language or HTML was created by Tim Berners-Lee in the late 1990's. HTML was initially developed at Conseil Européen pour la Recherche Nucléaire, with the english translation European Organization for Nuclear Research (CERN), was then developed in Internet Engineering Task Force (IETF), World Wide Web Consortium (W3C) and then to  Web Hypertext Application Technology Working Group (WHATWG). 

*HTML defines the composition of a web page using markup. A markup is an apporach wherein it decribes how the content should be displayed.


HTML Version history:

HTML:
> The prestandardization of HTML was limited. HTML tags were created to defined content for webpages. This inluded with 20 elements.
* Title tag, Next ID tag, Base Adress, Anchors, Sections, Paragraphs, Headings*

HTML 2.0:
This version was the standard during the rise of WWW. Along with the previous features of the first version, HTML 2.0 added new features like, Forms, Image, Line Break etc.

HTML 3.2:
> In 1997, HTML 3.2 was created with the agreement to remove decorative text attributes like <marquee> and <blink>. This was also the year where early browsers, IE and Netscapes, approved of this HTML version.

HTML 4.0 and 4.01:
> HTML 4.0 was launched in December 1997 with the name Cougar. HTML 4.0 supports the option given by the previous HTML versions, scripting languages, stylesheets. 
>HTML 4.0 was then approved to be HTML 4.01 that supports presentational elements, like fonts and colors.

XHTML:
> XHTML was created to provide interoperability with HTML and XML-based systems. This was also in lieu with the usage of XML tools in editing and creating.

HTML Document Structure
<!DOCTYPE html>
<html lang="en">

	<head>
		<meta charset="utf-8">
		<link rel="stylesheets" href="style.css">
		<title>HTML</title>
	</head>

	<body id="html-body">
		<!-- your intended content -->
	</body>

</html>

"DOCTYPE" - This declaration says to the browser what version of HTML the page is written in.
"html" - This tag tells the local browser that it is an HTML document. The html tag is with an attribute lang that specifies the language of the content of the page.
"head" - This tag states information about the HTML document. After the head tag, it is usually followed by the meta tag, wherein this tag says the character encoding (i.e uft-8). The first character encoding standard was ASCII. ASCII have 128 defined charaters, namely, English letters numbers, and special characters. 
"link" - this tag connects to a separate css file.
"title" - defines the title of your content.
"body" - this is where you write the whole content of your page.
