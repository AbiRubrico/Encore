Who made CSS?

Hakon Wium Lie proposed the concept of CSS while working with Tim-Berner Lee(Inventor of World wide web) in 1994. The first CSS released as a official

W3C(World Wide Web Consortium) on December 17, 1996. So, first officially CSS created by Hakon Wium Lie and Bert Bos (developed Argo web browser).
