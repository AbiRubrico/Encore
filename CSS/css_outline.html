CSS Outline

An outline is a line that is drawn just outside the border edge of the elements such as buttons, active form fields, etc., to make them stand out.

Outline-width Property

It specifies the width of an outline that to be added on a elements. The keyword used is "outline-width: (thick, thin or meduim)".

Outline-style Property

It sets the style for an outline like solid, dotted, dashed, double, groove, ridge, insert and outset.

Outline-color Property

It sets the color of the outline.

Outline Shorthand Property

In shorthand you can decraled all the properties in one line code by using single rule.
