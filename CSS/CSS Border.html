CSS Border

Border Style

The border-style property specifies what kind of border to display.

Here are some values are allowed:

  • dotted - Defines a dotted border.
  • dashed - Defines a dashed border.
  • hidden - Defines a hidden border.
  • solid - Defines a solid border.
  • double - Defines a double border.

    • Example: (Try editing it in your css file)

     p.double {border-style: double;}

    Border Color

    The border-color property is used to set the color of the four borders.

    Example: (Try editing it in your css file) 

     p.redborder {
  border-style: solid;
  border-color: red;
}

    Border Width

    The border-width property can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick.

    Example: (Try editing it in your css file) 

     p.px {
  border-style: solid;
  border-width: 5px;
}

p.word {
  border-style: solid;
  border-width: medium;
}
