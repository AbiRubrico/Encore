CSS Syntax

Selector - is a coude snippet used to identify the web page elements.

Property - is an aspect of the elements that is to be affected, like colors, padding, margin and background.

Value - it is used to define a property. For example: color: green;

Single Selector and Declaration block that follows the combination of ruleset.

At-Rules - it is similar to rulesets but it begins with @ sign. For example: @media

Comments - it is used to explain your codes, start by using /* and ends with */.

External Style Sheet - you can change the entire website by changing one file. It should be in the head section.

Internal Style Sheet - it can be used if one single page is a unique style. It can be defined within <style> element inside of <head> section. It should be in the head section.

Inline Style - it can be used to apply a unique style for a single element. It should be inside the HTML element.
